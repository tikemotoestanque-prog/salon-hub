# LINE公式アカウント 実機デモ 構築手順書（SalonHub商談用）

サロンオーナーに「実際のLINEでこう動きます」を見せるための、LINE公式アカウントのセットアップ手順です。
コピペできる文面つき。**所要 約60〜90分**。専門知識は不要です。

---

## ✅ これで作れるもの / ❌ 作れないもの（正直な線引き）

**✅ 標準機能だけでできる（このデモの範囲）**
- 友だち追加時の自動あいさつメッセージ
- リッチメニュー（予約・クーポン・マイカルテ等のボタン）
- ステップ配信（友だち追加◯日後に自動でフォローメッセージ）
- クーポン配信、一斉配信
- リッチメニューのボタンから **SalonHubの画面を開かせる**（＝LINE→SalonHubの流入を実機で見せられる）

**❌ 標準ではできない（＝Messaging API＋バックエンドが必要な次フェーズ）**
- お客さん「一人ひとり」に合わせて文面を自動で出し分け（カルテ連動）
- 来店データとLINE友だちの自動ひも付け
- SalonHubの顧客データを使った自動配信

> デモでは「**一律のステップ配信＋リッチメニュー**」までを実機で見せ、「ここから先、お客さんごとに自動で出し分けるのがSalonHubの本番です」と繋げるのが王道です。

---

## 🧰 準備するもの
- スマホ（自分のLINE）
- サロンのロゴ or 適当な画像（プロフィール用・正方形推奨）
- リッチメニュー用の画像（後述。Canva無料テンプレでOK）
- PC（リッチメニュー作成はPCが楽。スマホでも可）

---

## STEP 1. アカウント開設（無料）

1. PCで **LINE公式アカウント** の作成ページを開く → `https://www.lycbiz.com/jp/` の「アカウントの開設」
2. 自分のLINEアカウント（または LINEビジネスID）でログイン
3. アカウント情報を入力
   - **アカウント名**：`Hair Salon GRACE`（デモ名。本番は実店舗名）
   - **業種**：美容・ヘアサロン
4. プラン：**コミュニケーションプラン（無料・月200通まで）** を選択 → デモには十分
5. 作成完了 → **LINE Official Account Manager**（管理画面）に入る

---

## STEP 2. 基本プロフィールを整える（5分）

管理画面 →「設定」→「アカウント設定」「プロフィール」で：
- プロフィール画像（ロゴ）
- ステータスメッセージ：`✂️ 予約・ご相談はトークから。来店後はマイカルテとケア情報をお届けします`
- 営業時間：`10:00〜20:00（火曜定休）`
- ボタン（予約・電話など）を表示ONに

---

## STEP 3. あいさつメッセージ（友だち追加で自動送信）

管理画面 →「トークルーム管理」→「あいさつメッセージ」に、下記をコピペ。

```
友だち追加ありがとうございます！🌿
Hair Salon GRACE 公式LINEです。

▼ できること
・ご予約・ご変更はこのトークから24時間OK
・来店後はマイカルテ＆ホームケア情報を自動でお届け
・友だち限定クーポンを配信

まずは下のメニューから「ご予約」をどうぞ💁‍♀️
スタッフ一同、お会いできるのを楽しみにしています✨
```

> 💡 改行・絵文字はそのままで自然に見えます。1通で収めると印象が良いです。

---

## STEP 4. リッチメニュー（トーク下の固定ボタン）

管理画面 →「トークルーム管理」→「リッチメニュー」→「作成」

- **表示期間**：今日〜（終了日なしでOK）
- **テンプレート**：6分割（大）を選択
- **タイトル（管理用）**：`メインメニュー`
- **メニューバーのテキスト**：`メニューを見る`

**画像**：Canvaなどで6マスの画像を1枚作る（各マスにアイコン＋文字）。面倒なら無地＋文字でも可。

**各ボタンのアクション設定**（タップ領域ごと）：

| マス | ラベル | アクション（タイプ→設定） |
|---|---|---|
| ① ご予約 | 予約 | リンク → 予約ページのURL（後述・デモは下記） |
| ② メニュー・料金 | メニュー | リンク → デモHPのURL |
| ③ クーポン | クーポン | クーポン → 後述のクーポンを作成して紐付け |
| ④ マイカルテ | カルテ | **リンク → `https://salon-hub-plum.vercel.app/m/c001`** |
| ⑤ アクセス | アクセス | リンク → GoogleマップURL（任意） |
| ⑥ お問い合わせ | 相談 | テキスト → `ご相談内容をこちらにご記入ください😊` |

> ⭐ **デモの肝**：④マイカルテのリンクに **SalonHubのお客さん画面URL** を入れると、LINEのボタンから実際にSalonHubのマイカルテが開きます。これが「LINE → SalonHub」を実機で見せる決定打になります。
> ①予約は、デモなら `https://salon-hub-plum.vercel.app/lp`（デモHP）でも可。

---

## STEP 5. ステップ配信（来店フォローの自動化）

管理画面 →「ステップ配信」→「作成」
- **開始のトリガー**：友だち追加
- 下記3通を、それぞれ「◯日後に配信」で設定（無料プランの通数に注意）

**① 追加直後（0日後）**
```
さっそく友だち追加ありがとうございます😊
ご予約は下のメニュー「ご予約」からいつでもどうぞ。
気になることがあればこのトークでお気軽にご相談くださいね✂️
```

**② 1日後（来店お礼の想定）**
```
本日はありがとうございました！担当より🌿
仕上がりはいかがでしたか？
気になる点があれば、このトークからいつでもご相談ください。
お家でのスタイリングのコツもまたお送りしますね😊
```

**③ 14日後（ホームケア＆次回案内）**
```
その後、髪の調子はいかがですか？😌
仕上がりを長持ちさせるホームケアのコツです🧴
・洗い流さないトリートメントを毛先中心に
・乾かす前にしっかりタオルドライ

そろそろ次回のご予約もおすすめの時期です🗓
下のメニュー「ご予約」からどうぞ💁‍♀️
```

> 💡 本物は「来店日」を起点に送りたいですが、標準のステップ配信は「友だち追加」起点です。**お客さんごとに来店日を起点に出し分ける＝SalonHubの本番（次フェーズ）** という説明に繋げられます。

---

## STEP 6. クチコミ依頼（景表法・ステマ規制に注意）

- クーポンや特典と「クチコミ投稿」を **引き換えにしない**（Googleポリシー違反＋ステマ規制）。
- 王道は「依頼を自動化・簡単化」するだけ。下記を配信 or リッチメニューに：

```
いつもありがとうございます🙏
もしよろしければ、Googleでご感想をいただけると
スタッフのとても励みになります！
（無理のない範囲で大丈夫です😊）
👉（GoogleクチコミのURL）
```

---

## STEP 7. SalonHubとの繋ぎ方（デモでの見せ方）

実機デモはこの流れが一番伝わります：

1. **デモHP**（`/lp`）を見せる →「LINEで予約」をタップ
2. （本番は）友だち追加 → **あいさつメッセージが自動で届く**
3. リッチメニューの **「マイカルテ」をタップ → SalonHubのお客さん画面が開く**
4. オーナーに：「お客さんにはLINEがこう見えます。**裏でSalonHubが、この人だけに合わせて自動でメッセージを出し分けます**」と SalonHubのオーナー画面（ステップ配信の裏側）を見せる
